What is the difference in among Map, filter, find and forEach?

Map : Here map is a arrow function. We can play it on array method It is a impression function having a variable that will give us a return that's why we can use this in another task

forEach : Here forEach is a arrow function. We can play it on array method It is a impression function without having a variable

Filter : Here filter is also a arrow function having variable. We can play it on array method.There is condition after arrow sign and which are true will give us a return that's why we can use the result in another task.

Find : Here find is also a arrow function having variable. We can play it on array method.There is condition after arrow sign and which one (first one) true will give us a return that's why we can use the result in another task.

What is a difference in among LocalStorage, SessionStorage and Cookies?

LocalStorage : Here LocalStorage means a storage in web. where we can store data and that will forever access able without expiration date. We can use it whenever we need it also.

SessionStorage : SessionStorage is also a kind of a storage system on web most like LocalStorage but difference SessionStorage is the data cleared when web tab(page session) close

Cookies : Cookies is a tricks played by a website when a user visit, the website takes user's activities' information if user permit and stored in LocalStorage.
